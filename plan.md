# 大会内容
デジ窓フェスティバル（デジフェス）という、スプラトゥーン3を用いたナワバリトーナメントとエリアトーナメントを実施する大会。

# 参照データ
data/申請.csv のチーム一覧

# 表示したい内容
NodeCG Control
ナワバリ、エリアの2つのWorkspaceを作成し、それぞれに以下のパネルを設置する。
* チーム選択パネル
  * アルファチーム選択プルダウン
  * ブラボーチーム選択プルダウン

* ボタンパネル
  * アルファチーム表示ボタン
  * ブラボーチーム表示ボタン
  * リセットボタン（チーム選択を初期化、チーム表示を非表示）

* プレビューパネル
  * アルファ、ブラボーそれぞれ、チーム選択パネルで選択したチームの以下情報が記載される。
    * 二つ名
    * チーム名
    * プレイヤー1～4
  * 各行には編集ボタンがあり、押すと編集できるようにする。編集後の内容がGraphicに反映される。

また、設定 Workspaceを作成し、CSVを再読み込みするボタンを設置する。

NodeCG Graphic
ナワバリ、エリアそれぞれで、配信画面下側の表示（[Turf-War|Splat-Zones]-under）と、会場画面左右用の表示（[Turf-War|Splat-Zones]-side）のGraphicを用意する。
Controlの表示ボタンを押したら、underとsideが0.5秒のフェードインで表示されるようにする。
Controlのリセットボタンを押したら、underとsideが0.5秒のフェードアウトで消える。

すべて1920x1080の表示領域を想定。sideは会場の映像機器で960x1080に左右分割する。
以下に仕様を示すが、文字の表示は大まかに配置してもらって、細かいレイアウトやフォントサイズはCSSで手動で調整したい。
装飾もCSSで手動で行うので、クラスだけ設定しておいて文字の装飾は一旦行わないこと。

under仕様
表示領域はアルファチームが左下で960x70、ブラボーチームが右下で960x70。
どちらも表示領域内で中央寄せ。
上側にチーム名、下側にプレイヤー名を全角スペース区切りで表示する。
フォントの大きさはチーム名とプレイヤー名1:1で。

side仕様
表示領域はアルファチームが左上で960x960、ブラボーチームが右上で960x960。
アルファチームは左寄せ、ブラボーチームは右寄せ。
上から二つ名、チーム名、プレイヤー名1～4を表示する。
チーム名が長い場合2行に改行する。改行箇所の提示はCSV内に記載したいので、簡素な方法を最初に提案してほしい。
チームを切り替えた時に、チーム名の改行などがあってもプレイヤー名の場所は変わらないような配置方法にする。
フォントの大きさは一旦、二つ名:チーム名:プレイヤー名=1:3:2で。
